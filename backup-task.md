# Backup Task

The purpose of this task is to back up server data to an off-site location.

This is important because off-site backups protect against data loss from hardware failure, accidental deletion, or disaster.

## Steps

1. [ ] Identify data to back up
1. [ ] Transfer data to remote storage using [rsync.net](rsync-net-tool.md)
1. [ ] Verify backup completed successfully
